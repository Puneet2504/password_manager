## Note 
### Before going to authenticate just make sure that you have saved your face data in the face biometric settings. Otherwise no effect will be seen on the Face Authentication Page.
### This app works fine for Samsung devices with face recognition biometric.
### This android app will  not work on devices that prefer fingerprint biometrics over face recognition.
